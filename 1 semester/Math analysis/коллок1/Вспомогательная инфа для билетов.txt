БИЛЕТ №3
1) Для каждой пары чисел существуют только такие соотношения: {=, <, >}. Знак < обладает транзитивностью. Если a < b, то найдётся такое c, что a < c < b.
2) Сложение и вычитание: ассоциативность и коммутативность для сложения; a + 0 = a; a + (-a) = 0; если a < b, то для любого с: a +- c < b +- c; Разность a - b единственна, и равна a + (-b), т.к. если прибавить к разности число b, то получается a, как и если это прибавить к выражению a + (-b), ведь это будет (a + (-b)) + b = a + ((-b) + b) = a + 0 = a
3) Умножение и деление: ассоциативность и коммутативность для умножения; a * 1 = a; a * (1/а) = 1, для a != 0; дистрибутивность; если a < b, то для любого положительного с: ac < bc; Частное a/b, для b != 0, единственно, и равно a * (1/b), т.к. если умножить на частное число b, то получается a, как и если его умножить на выражение a * (1/b), ведь это будет ( a * (1/b)) * b = a * ((1/b) * b) = a * 1 = a
4) Архимедово свойство про существование натурального числа, которое больше, чем заданного действительное c; и про существование натурального числа, обратное значение которого меньше, чем заданное положительное c.
5) Если последовательность действительных чисел {An} не убывает и ограничена сверху числом M, то существует число A <= M, к которому эта последовательность стремится, как к своему пределу.
БИЛЕТ №4
Сегменты = отрезки
Аксиома о непрерывности множеств действительных чисел: Если есть 2 множества, и каждый элемент первого множества не больше каждого элемента второго множества, то найдётся действительное число, которое не меньше всех элементов первого множества и не больше всех элементов второго множества.
Лемма из билета: Для всякой системы вложенных отрезков ( |A1,B1|⊃|A2,B2|⊃...⊃|An,Bn|⊃... ) существует хотя бы одна точка c, которая принадлежит всем отрезкам данной системы.
Док-во: Докажем, что для любой пары натуральных чисел k и l, работает, что Ak<=Bl. Если k>=l: Ak<=Al<=Bl. Если k<l: Ak<=Bk<=Bl. Доказали. А теперь используем аксиому о непрерывности множеств действительных чисел на множества {A1,A2,...} и {B1,B2,...} и обнаружим, что существует хотя бы одно такое число c, которое находится между любой парой Ak и Bl, а соответственно и точка с такой координатой лежит на каждом отрезке этой системы.
Дополнение к лемме из билета: если последовательность, состоящая из разностей между концами отрезка, стабилизируется к нулю, то точка, принадлежащая сразу всем отрезкам только одна.
Докажем от противного. То, что у нас есть хотя бы одна такая точка, мы доказали. Но вот что, если их реально хотя бы 2? Возьмём эти 2 точки и их разность. Пусть это некое число E (E>0). Т.к. последовательность длин отрезков стабилизируется к нулю, то начиная с какого-то n0-го отрезка разница между длинами всех отрезков и нулём, станет меньше Е, то есть длины каждого из таких отрезков будут меньше Е, а соответственно такие отрезки не могут вмещать сразу 2 точки, расстояние между которыми Е, значит противоречие.
БИЛЕТ №5
Точная верхняя и нижняя грани множества - супремум(sup) и инфинум(inf)
Просто грань - любое число, ограничивающее множество, то есть такое число, которое больше или меньше всех элементов множества.
А вот точная грань, это уже крайнее такое число. То есть sup - минимальная из всех верхних граней, а inf - максимальная из всех нижнех граней.
Теорема о существовании точной верхней грани *это также и работает для нижней, просто нужно проинверсировать некоторые части теоремы*: Непустое множество действительных чисел, ограниченное сверху, имеет точную верхнюю грань.
Док-во: Построим множество B из всех верхних граней множества А. Множество B непустое, т.к. сказано, что А ограничено сверху, то есть имеет хотя бы одну верхнюю грань. Значит любой элемент из А не больше, чем любой элемент из B, значит по аксиоме о непрерывности множеств действительных чисел, существует такое число с, что не больше всех элементов из B, и не меньше всех элементов из А. Оно и явялется единственным супремумом. А единственным, потому что, будь другое такое число, то одно из них было бы минимальным и мешало бы второму быть супремумом.

Необходимое условие сходимости: последовательность должна быть ограниченной.

БИЛЕТ №7
1) Теорема о сохранении знака последовательности: если предел последовательности не равен нулю, то существует такой индекс n0, что для всех n>=n0, знак An совпадает со знаком предела.
Док-во: Пусть предел этой последовательности равен А, возьмём E=|A|>0, т.к. A!=0. Значит найдётся такой индекс n0, начиная с которого все элементы отличаются от A не больше, чем на E=|A|, то есть того же знака, что и А, т.к. |An-A|<|A|, то есть An удалён от A не дальше, чем расстояние от А до 0, значит 0 не может быть между An и A.
2) Предельный переход в неравенствах: Если существуют 2 последовательности {An} и {Bn}, которые имеют пределы A и B, и найдётся такой индекс n0, что для всех n>=n0,  An < Bn, то A <= B.
Док-во: предположим обратное, то есть, что A > B. Тогда возьмём E = (A-B)/2 > 0. Значит начиная с какого-то n0 = max(n0({An}),n0({Bn})), для всех n>=n0, |An-A|<E и |Bn-B|<E, значит |A-An|+|Bn-B|<2E, и т.к. сумма модулей больше модуля суммы, то |A-An + Bn-B|<(A-B) <=> |A-B + Bn-An|<(A-B), значит A-B + Bn-An < (A-B), значит A-B < A-B, т.к. An < Bn, то есть противоречие.
3) Теорема о трех последовательностях: Если существуют 3 последовательности {An}, {Bn} и {Cn}, что для любого n∈N: An <= Bn <= Cn, а так же последовательности {An} и {Cn} имеют одинаковый предел B, то {Bn} тоже имеет предел, и он тоже равен B.
Док-во: для любого E>0 можно найти такое n0 = max(n0({An}),n0({Cn})), что для любого n>=n0: |An-B|<E и |Сn-B|<E, и т.к. An <= Bn <= Cn, то An-B <= Bn-B <= Cn-B, значит |Bn-B|<E => {Bn} стремится к B.
БИЛЕТ №8
1) lim(An +- Bn) = lim(An) +- lim(Bn).
Док-во: для любого E>0 можно найти такой индекс n0 = max(n0({An}),n0({Bn})), что для всех n>=n0 выполняется условие, что |An - lim(An)| < E/2 и |Bn - lim(Bn)| < E/2
|An - lim(An)| +- |Bn - lim(Bn)| < E => |An - lim(An) +- Bn -+ lim(Bn)| < E => |(An +- Bn) - (lim(An) +- lim(Bn))| < E. Значит lim(An +- Bn) = lim(An) +- lim(Bn).
2) lim(An*Bn) = lim(An)*lim(Bn).
Док-во: Выберем такое число М = max(sup{|Bn|}, |lim(An)|). Для любого E>0 можно найти такой индекс n0 = max(n0({An}), n0{Bn})), что для всех n>=n0 выполняется условие, что |An - lim(An)| < E/2M и |Bn - lim(Bn)| < E/2M, значит |Bn|*|An - lim(An)| < E/2 и |lim(An)|*|Bn - lim(Bn)| < E/2 => |Bn*(An-lim(An)) + lim(An)*(Bn-lim(Bn))| < E => |Bn*An - Bn*lim(An) + Bn*lim(An) - lim(An)*lim(Bn)| < E <=> |Bn*An - lim(An)*lim(Bn)| < E. Значит lim(An*Bn) = lim(An)*lim(Bn).
3) lim(An / Bn) = lim(An) / lim(Bn), при lim(Bn) != 0.
Док-во: Возьмём 2 последовательности: {an} и {bn}, что для натурального индекса k: ak=Ak-lim(An) и bk=Bk-lim(Bn). An/Bn = (lim(An)+an)/(lim(Bn)+bn) - lim(An)/lim(Bn) + lim(An)/lim(Bn) = (lim(An)*lim(Bn)+an*lim(Bn))/((lim(Bn)+bn)*lim(Bn)) + lim(An)/lim(Bn). Т.к. {an} и {bn} стремятся к нулю, то an и bn - беск. малые величины, т.е. (lim(An)*lim(Bn) + an*lim(Bn))/((lim(Bn)+bn)*lim(Bn)) = 0, а значит An/Bn = lim(An)/lim(Bn), т.е. lim(An / Bn) = lim(An) / lim(Bn). Доказательство стрёмное, но я бы не хотел долго сидеть над этим.

Док-во: x - б.м., y - огранич. => существует M > 0, что |y| < M. Для любого E>0: |x| < E/M. |x*y| = |x| * |y| < E/M * M = E => x * y → 0 (!)

БИЛЕТ №15
Частичный предел  последовательности - предел ее подпоследовательности.
Верхний предел (lim sup) последовательности - наибольший частичный.
Свойство: lim sup (xn + yn) ⩽  lim sup (xn) +  lim sup (yn)
Нижний предел (lim inf) последовательности - наименьший частичный.
Свойство: lim inf (xn + yn) ⩾  lim inf (xn) +  lim inf (yn)
Критерий сходимости: lim(An) = A <=> существует lim inf (An) = A, существует lim sup (An) = A
Доказательство: Предположим, что это не так. Пусть нам известно, что lim inf(An)= lim sup(An) = A, тогда докажем, что lim(An)= A.
Пусть дано E > 0, тогда существует n0 такое, что для всех n >= n0: Ak > A - E, при k >= n. Также существует N0, что для всех n >= N0: sup Ak < A + E, при k >= n. Возьмем N = max(N0, n0) , N0 и n0 - натуральные числа. Тогда: A - E < An < A + E <=> |An - A| < E. Значит lim(An) = A.

Утверждение о предельной точке ограниченного множества: х0 - предельная точка множества E, E - бесконечное ограниченное множество, тогда существует последовательность {xn}, состоящая из элементов этого множества, что lim(xn)= x0
Доказательство утверждения: существуют такие x1 ≠ x0 , x2 ≠ x0 , x3 ≠ x0 ,…, xn ≠ x0. Множество E - ограниченное => существуют M и m ∈ E, что ∀x∈Е : m ⩽ x ⩽ M => ∀xn: m ⩽ xn ⩽ M ∀n ∈ N, {xn} - ограниченная последовательность => (по т. Больцано-Вейерштрасса)  существует {xnk} → а, значит а - предельная точка.
БИЛЕТ №17
Функция - соответствие между элементами двух множеств, установленное по такому правилу, что каждому элементу первого множества соответствует один и только один элемент второго множества.
Предел функции:
1) Значение А называется пределом функции f(x) в точке x0, если для любого E>0 найдётся δ такое, что для всех аргументов x, удовлетворяющих условию 0 < |x-x0| < δ выполняется неравенство |f(x) - A| < E (Определение по Коши)
2) Значение A называется пределом функции f(x) в точке x0, если для любой окрестности O(A) точки A существует проколотая окрестность O(x0) точки x0 такая, что образ этой окрестности f(O(x0)) лежит в O(A) (Ещё одно определение по Коши)
3) Значение А называется пределом функции f(x) в точке x0, если для любой последовательности точек, сходящейся к x0, но не содержащей x0 в качестве одного из своих элементов, последовательность значений функции сходится к A (Определение по Гейне)
Существование пределов: lim(x→x0-0)(f(x)) = A <=> ∀ε>0, ∃δ(ε), ∀x ∈Е x∈(x0-δ, x0) : |f(x) - A| < ε; lim(x→x0+0)(f(x)) = A <=> ∀ε>0, ∃δ(ε), ∀x ∈ Е x ∈ (x0, x0+δ) :    |f(x) - A| < ε; lim(x→x0)(f(x)) = A <=> ∃lim(x→x0-0)(f(x)) = A и ∃lim(x→x0+0)(f(x)) = A
